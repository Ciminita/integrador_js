Variables en JavaScript

¿Qué son las variables?

Las variables en JavaScript como en otro lenguajes de programación, es un contenedor para un valor, como un número que podríamos usar en una suma, o una cadena que podríamos usar como parte de una oración. Pero una cosa especial acerca de las variables es que los valores que contienen pueden cambiar. En cambio una constante son valores que no se pueden cambiar.

¿Cuál es la diferencia entre var, let y const?

Cuando hablamos de var es como la versión anterior para crear variables. En cambio cuando hablamos de let y const son las nuevas formas de crear variables y en las últimas versiones de cualquier navegador ya son soportadas.

La posibilidad de crear una variable con let no permite volver a declarar variables pero si cambiar su valor. Esta es la principal diferencia con var, que si permite re declarar la variable de nuevo. Cuando declaramos una variable con const, la variable no puede apuntar a otro elemento que no sea el inicialmente definido.
